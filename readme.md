# Работа с Git
---
## Локальный репозиторий.
* Создаём директорию проекта.
* Создаём файлы проекта.
* Инициализируем репозиторий
```bash
git init
```
* Проверяем состояние репозитория
```bash
git status
```
* Добавляем файлы в _отслеживание_.
```bash
git add --all
```
* Делаем commit.
```bash
git commit -m "Описание"
```
